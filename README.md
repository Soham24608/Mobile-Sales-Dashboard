**📊 Mobile Sales Dashboard – Power BI Project**
📌 Project Overview

The Mobile Sales Dashboard is an interactive Business Intelligence dashboard built using Power BI to analyze mobile phone sales performance across multiple dimensions such as time, location, brand, model, and payment methods.

This project demonstrates the use of Power Query for data transformation, custom calendar creation, and advanced Power BI visualizations to generate meaningful business insights.

The dashboard enables users to:

Monitor total sales performance

Analyze sales trends over time

Compare sales across cities and brands

Evaluate customer transactions and payment methods

Track Month-To-Date (MTD) performance

🎯 Project Objectives

Transform raw sales data into a structured analytical model

Create a custom calendar table to analyze data by:

Year

Quarter

Month

Day

Build an interactive Power BI dashboard for business insights

Provide data-driven insights for decision-making

🔧 Data Preparation & Transformation

Data preprocessing was performed using Power Query to ensure accuracy and usability.

Key Transformation Steps

1️⃣ Data Cleaning

  Removed null and duplicate values

  Standardized column names and formats

2️⃣ Data Type Conversion

  Converted date fields to proper Date format

  Ensured numeric fields such as sales, quantity, and price were properly typed

3️⃣ Feature Engineering

  Created calculated columns and measures to support analysis

4️⃣ Custom Calendar Table
  A custom calendar table was created to enable advanced time-based analysis such as:

  Year

  Quarter

  Month

  Day

  This allows the dashboard to perform time intelligence calculations such as:

  Month-To-Date (MTD)

  Year-To-Date comparisons

  Trend analysis
